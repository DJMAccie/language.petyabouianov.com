## Design Context

### Users
- Primary user is the site owner using the app as a personal language-study workspace for Japanese travel prep and ongoing study.
- Sessions happen in short, focused bursts on desktop and mobile, so core interactions need to stay fast, scannable, and low-friction.
- Reference material such as grammar patterns and lesson sentences should be easy to access without cluttering the main review flow.

### Brand Personality
- Calm, focused, personal.
- More like a lightweight Finder-style study utility than a gamified language product.
- The interface should support concentration and confidence rather than excitement or novelty.

### Aesthetic Direction
- Light-first glass-window chrome over a photographic background.
- Compact controls, muted neutrals, restrained accent colors, and table/list-first layouts.
- Prefer stacked rows, separators, and clean density over card grids or dashboard styling.
- Avoid feature-heavy copy, decorative clutter, and rounded-card interfaces that break the app's existing rhythm.

### Design Principles
- Keep primary study flows scannable and list-first.
- Match new surfaces to the studio's existing visual language instead of inventing a separate subsystem.
- Favor concise copy, clear labels, and obvious actions.
- Support long text, mobile use, and quick context switching without visual noise.
- Add polish through spacing, alignment, and hierarchy rather than extra decoration.
